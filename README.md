# Synth AI — Demo Site

> Portfolio demonstration project. All content is fictional, no real product or client data was used.

## Overview
A high-converting SaaS landing page for a fictional AI writing assistant, 
built to showcase frontend development skills in a modern dark-mode design.

## Features
- Responsive design (mobile & desktop)
- Mobile hamburger menu
- Sticky navbar with scroll effect
- Hero section with typewriter animation
- Glassmorphism card design
- Scroll reveal animations (Intersection Observer)
- Pricing section with monthly/yearly billing toggle
- Testimonials section with avatars
- Structured Data (Schema.org) for SEO
- Dynamic footer year

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Inter)
